# Telegram Bot for LETI Schedule

Бот для получения расписания занятий СПбГЭТУ "ЛЭТИ"

## Функции
- Расписание на сегодня/завтра
- Вся неделя
- Конкретный день и неделя
- Ближайшее занятие

## Установка
1. `pip install -r requirements.txt`
2. Создай файл `.env` с токеном бота
3. `python main.py`

## Команды
- `/start` - начать
- `/today [группа]` - на сегодня
- `/week [группа]` - вся неделя
- и т.д.
